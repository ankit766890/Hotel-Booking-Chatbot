# 🏨 Hotel Booking Chatbot (Flask)

An interactive and lightweight **Web-based Chatbot** designed to handle hotel room inquiries and bookings. This project is built using a **Python Flask** backend and a responsive **HTML/JS** frontend.

---

## ✨ Features

* **💬 Smart Interaction:** Instant responses to user queries using keyword matching.
* **🛌 Room Inquiries:** Check availability for Single, Double, and Triple rooms.
* **💰 Pricing Details:** Get up-to-date pricing for different room categories.
* **🕒 Live Updates:** Provides current system date and time upon request.
* **🏨 Hotel Info:** Details on facilities (WiFi, Parking, Room Service), location, and contact info.
* **✅ Instant Confirmation:** Automated booking confirmation based on user selection.

---

## 🛠️ Tech Stack

* **Backend:** [Python 3.12](https://www.python.org/)
* **Framework:** [Flask](https://flask.palletsprojects.com/)
* **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
